# Crear tu primera productora

<Card title="¿Qué es una productora?" icon="circle-info" href="/organizations/index" cta="Click aquí">
  Más detalles sobre productoras
</Card>

<Steps>
  <Step title="Ir a crear productora">
    Panel de Control → "Crear" en "Mis Productoras"
  </Step>
  <Step title="Completar información">
    - **Nombre:** Ej: "Eventos López"
    - **País:** selecciona tu país del listado
    - **Teléfono:** Para contacto de soporte
    - **Logo:** Opcional

    <img
      src="/images/qrticket.app_dashboard.png"
      alt="Qrticket App Dashboard Pn"
      title="Qrticket App Dashboard Pn"
      style={{ width:"49%" }}
    />
  </Step>
  <Step title="¡Listo!">
    Recibís 30 créditos gratis automáticamente
  </Step>
</Steps>

## Después de crear

Desde el menú de tu productora podés:

<CardGroup cols={2}>
  <Card title="Administrar tus eventos" icon="calendar">
    Organizá tus eventos desde el menú principal
  </Card>
  <Card title="Comprar créditos" icon="coins">
    1 crédito = 1 entrada generada
  </Card>
  <Card title="Conectar MercadoPago" icon="credit-card">
    Para ventas automáticas online
  </Card>
  <Card title="Agregar equipo" icon="users">
    Invitá colaboradores con roles específicos
  </Card>
</CardGroup>
