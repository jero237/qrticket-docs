# Crear tu primera productora

Una **productora** es como una "carpeta" que agrupa todos tus eventos y comparte créditos entre ellos.

<Card title="¿Qué es una productora?" icon="info">
  Más detalles sobre productoras → [Ver guía completa](/organizations/index)
</Card>

## Crear en 3 pasos

<Steps>
  <Step title="Ir a crear productora">
    Panel de Control → "Crear" en "Mis Productoras"
  </Step>
  <Step title="Completar información">
    - **Nombre:** Ej: "Eventos López" 
    - **País:** selecciona tu país del listado
    - **Teléfono:** Para contacto de soporte 
    - **Logo:** Opcional
  </Step>
  <Step title="¡Listo!">
    Recibís 30 créditos gratis automáticamente
  </Step>
</Steps>

## Después de crear

Desde el menú de tu productora podés:

<CardGroup cols={2}>
  <Card title="Crear eventos" icon="calendar">
    Organizá tus eventos desde el menú principal
  </Card>
  <Card title="Comprar créditos" icon="coins">
    1 crédito = 1 entrada generada
  </Card>
  <Card title="Conectar MercadoPago" icon="credit-card">
    Para ventas automáticas online
  </Card>
  <Card title="Agregar equipo" icon="users">
    Invitá colaboradores con roles específicos
  </Card>
</CardGroup>

## Configuración recomendada

<AccordionGroup>
  <Accordion title="1. Conectar MercadoPago (opcional)">
    Para ventas online automáticas → [Guía MercadoPago](/organizations/mercadopago)
  </Accordion>
  <Accordion title="2. Agregar equipo (opcional)">
    Para trabajar con colaboradores → [Gestión de equipo](/organizations/team)
  </Accordion>
  <Accordion title="3. Comprar más créditos">
    Si necesitás más de 30 → [Comprar créditos](/credits/credits-purchase)
  </Accordion>
</AccordionGroup>
