# Productos del evento

Los **productos** son todo lo que podés vender en tu evento: entradas de acceso, **barra** (tragos y bebidas que se canjean con QR) y combos que combinan varios elementos.

<Card title="1 crédito = 1 producto" icon="coins">
  Cada vez que se genera o vende cualquier producto (entrada o producto de barra), se descuenta 1 crédito.
</Card>

## Tipos de productos

<CardGroup cols={3}>
  <Card title="Entradas" icon="ticket">
    **Acceso al evento**

    Dan derecho de ingreso. Ejemplos:

    - General, VIP, Preventa
    - Estudiantes, Jubilados
    - Anticipada, En puerta
  </Card>
  <Card title="Barra" icon="martini-glass" href="/events/barra">
    **Pre-venta de tragos y bebidas**

    Tus clientes las compran online y las canjean en la barra con su QR:

    - Cerveza, Fernet, Gin tonic
    - Combos de tragos
    - Agua, gaseosa, energizante
  </Card>
  <Card title="Combos" icon="box">
    **Paquetes combinados**

    Combinan entradas \+ barra:

    - Entrada \+ 2 cervezas
    - 2x1 entrada General
    - Entrada VIP \+ consumo
  </Card>
</CardGroup>

## Crear productos

### Paso 1: Acceder a productos

<Steps>
  <Step title="Desde tu evento">
    Dashboard → Tu evento → "Productos" → "Crear"
  </Step>
  <Step title="Elegir tipo">
    Seleccioná: **Entrada**, **Barra** o **Combo**
  </Step>
</Steps>

### Paso 2: Configurar el producto

Tomando una **Entrada** como ejemplo, estos son los campos del editor:

<AccordionGroup>
  <Accordion title="Información básica">
    **Requerido:**

    - **Nombre:** Ej: "General", "Cerveza", "VIP \+ Cena"
    - **Estado:** Activo, Oculto, Exclusivo, Agotado o Inactivo (ver más abajo)

    **Opcional:**

    - **Descripción:** Detalles adicionales para compradores
  </Accordion>
  <Accordion title="Precio">
    - **Con precio:** Se cobra en las ventas por MercadoPago, Stripe o transferencia bancaria.
    - **Sin precio (vacío):** El producto es gratuito. Cualquier persona con el link del evento podrá reclamarlo.

    <Tip>
      Si dejás el precio vacío, el editor te avisa: "Este producto es gratuito y está activo. Cualquier persona con el link del evento podrá reclamarlo."
    </Tip>
  </Accordion>
  <Accordion title="Límites y vigencia (opcionales)">
    - **Límite por Compra:** Máximo de unidades que alguien puede llevar en una misma compra.
    - **Límite:** Total disponible del producto (ej: 200 entradas VIP). Cuando se alcanza, el producto pasa a "Agotado".
    - **Limitar vigencia de uso:** Activá este checkbox para restringir en qué fechas o momentos se puede usar el producto.
  </Accordion>
  <Accordion title="Estados del producto">
    - **Activo:** Visible para todos.
    - **Oculto:** Equipo de la productora y link directo.
    - **Exclusivo:** Solo administradores.
    - **Agotado:** Sin disponibilidad.
    - **Inactivo:** Desactivado.
  </Accordion>
</AccordionGroup>

## La lista de productos

En **Productos** vas a ver tus productos organizados en pestañas: **Tipos de Entrada**, **Barra** y **Combos**.

<Note>
  La pestaña **Barra** tiene su propia guía, con presets de bebidas, link de barra y afiche QR. Mirá [Barra (pre-venta de tragos)](/events/barra).
</Note>

La tabla muestra estas columnas:

| Columna | Qué muestra |
| --- | --- |
| **Nombre** | El nombre del producto |
| **Estado** | Activo, Oculto, Exclusivo, Agotado o Inactivo (lo cambiás desde acá) |
| **Precio** | El precio configurado (o gratuito si está vacío) |
| **Límite** | Total disponible del producto |
| **Vendidas** | Cuántas unidades se vendieron |
| **Recaudado** | El total recaudado por ese producto |

<Tip>
  Con el botón **Exportar** descargás la lista de productos para revisarla por fuera de la plataforma.
</Tip>

## Gestión de productos existentes

### Editar productos

<Steps>
  <Step title="Seleccionar producto">
    Hacé clic en el nombre del producto que querés editar
  </Step>
  <Step title="Modificar configuración">
    Podés cambiar: nombre, descripción, estado, límites y la vigencia de uso
  </Step>
  <Step title="Guardar cambios">
    Los cambios son inmediatos y afectan las ventas futuras
  </Step>
</Steps>

<Warning>
  El **precio no se puede modificar** después de crear el producto: *"El precio no se puede modificar después de crear el producto. Creá uno nuevo para cambiarlo."* Los productos ya vendidos no se ven afectados por los demás cambios.
</Warning>

## Problemas comunes

<AccordionGroup>
  <Accordion title="'No tengo suficientes créditos para generar entradas'">
    **Causa:** Balance de créditos insuficiente para la demanda

    **Solución:** Comprar más créditos antes de que se agoten las ventas
  </Accordion>
  <Accordion title="'Los compradores no ven un producto'">
    **Verificá:**

    - Que el estado sea **Activo** (Oculto solo es visible para el equipo y por link directo; Exclusivo solo para administradores)
    - Que no esté **Agotado** (se alcanzó el límite)
    - Que no esté **Inactivo**
  </Accordion>
  <Accordion title="'Quiero cambiar el precio de un producto'">
    El precio **no se puede editar** una vez creado el producto. Creá un **producto nuevo** con el precio actualizado y, si querés, pasá el viejo a estado **Inactivo** u **Oculto**.
  </Accordion>
</AccordionGroup>
