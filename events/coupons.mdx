# Cupones de descuento

Los **cupones de descuento** son códigos que tus compradores aplican en la compra web del evento para pagar menos. Ejemplo: `PREVENTA20` da un 20% de descuento.

<Note>
  Estos cupones son **distintos** de los [cupones promocionales de la productora](/credits/coupons): esos te dan créditos gratis, en cambio los de acá son descuentos que tus compradores usan al comprar entradas en la web de tu evento.
</Note>

## Cómo acceder

<Steps>
  <Step title="Ir a tu evento">
    Dashboard → Tu evento
  </Step>
  <Step title="Abrir 'Cupones'">
    En el menú del evento, hacé clic en **"Cupones"**
  </Step>
</Steps>

## Lista de cupones

Vas a ver una tabla con todos los cupones del evento:

| Columna | Qué muestra |
| --- | --- |
| **Código** | El texto que ingresa el comprador |
| **Descuento** | Porcentaje o monto fijo del cupón |
| **Usos** | Cuántas veces se usó (y el límite, si tiene) |
| **Vencimiento** | Fecha de expiración, si la definiste |
| **Estado** | Si el cupón está activo o no |

Si todavía no creaste ninguno, vas a ver el mensaje **"No hay cupones creados para este evento."** Hacé clic en **"Crear"** para empezar.

## Crear un cupón

<Steps>
  <Step title="Abrir el formulario">
    En la sección "Cupones" del evento, hacé clic en **"Crear"**
  </Step>
  <Step title="Definir el código">
    Escribí el **Código** del cupón. Solo se permiten **letras, números, guión y guión bajo**. Si no se te ocurre uno, usá el botón para **generar un código aleatorio**.
  </Step>
  <Step title="Elegir el tipo de descuento">
    Seleccioná **Porcentaje (%)** o **Monto fijo**.
  </Step>
  <Step title="Cargar el valor">
    - Si elegiste porcentaje: ingresá el **Porcentaje de descuento (%)**
    - Si elegiste monto fijo: ingresá el **Monto** a descontar
  </Step>
  <Step title="Definir el límite de usos (opcional)">
    Ingresá un **Límite de usos** para que el cupón deje de funcionar al alcanzarlo. **Dejalo vacío para usos ilimitados.**
  </Step>
  <Step title="Definir el vencimiento (opcional)">
    Marcá el checkbox **"Tiene fecha de vencimiento"** y elegí hasta cuándo querés que el cupón siga activo.
  </Step>
  <Step title="Crear el cupón">
    Hacé clic en **"Crear Cupón"** para guardarlo.
  </Step>
</Steps>

## Cómo se aplica el descuento

El cupón se aplica en el **checkout de las ventas web** de tu evento. Cuando un comprador ingresa un código válido al pagar, el descuento se resta del total antes de completar la compra por **MercadoPago**, **Stripe** o **transferencia bancaria**.

<Tip>
  Compartí el código en tus redes o por WhatsApp para promocionar preventas, descuentos por tiempo limitado o beneficios para grupos.
</Tip>

<Card title="Configurar las ventas web" icon="gear" href="/events/sales">
  Revisá los canales de venta de tu evento para que el descuento funcione en el checkout.
</Card>
